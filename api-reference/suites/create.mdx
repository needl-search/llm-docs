---
title: "Create Suite"
api: "POST https://llm.needl.tech/suites"
description: "This endpoint creates a new test suite"
---

### Body

<ParamField body="name" type="string">
  The display name for this test suite.
</ParamField>
<ParamField body="required_inputs" type="string[]">
  The input variables required for any test cases added to this suite
</ParamField>

<RequestExample>

```bash Example Request
curl --location --request POST 'https://llm.needl.tech/suites' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer <token>' \
--data-raw '{
    "name": "Q & A Prompts",
    "required_inputs": ["documents", "question"]
}'
```

</RequestExample>

<ResponseExample>

```json Example Response
{
  "id": 1,
  "org_id": "000000-00000-0000-000000",
  "created_time": "00:00:00T00:00",
  "name": "Q & A Prompts",
  "required_inputs": ["documents", "question"]
}
```

</ResponseExample>
