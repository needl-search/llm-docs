---
title: "Update Suite"
api: "PATCH https://llm.needl.tech/suites/{suite_id}"
description: "This endpoint updates an existing suite"
---

### Body

<ParamField body="suite_id" type="number">
  The ID of the suite to be updated
</ParamField>

<ParamField body="name" type="string?">
  The updated name of the suite, if it should be changed
</ParamField>

<ParamField body="required_inputs" type="string[]?">
  The updated required_inputs of the string, if they should be changed
</ParamField>

### Response

<ResponseField name="id" type="number">
  ID of the suite
</ResponseField>

<ResponseField name="org_id" type="string">
  Organization ID the suite belongs to
</ResponseField>

<ResponseField name="created_time" type="string">
  ISO formatted time the suite was created
</ResponseField>

<ResponseField name="name" type="string">
  Display name of the suite
</ResponseField>

<ResponseField name="required_inputs" type="string[]">
  A list of the required inputs of the suite
</ResponseField>

<RequestExample>

```bash Example Request
curl --location --request PATCH 'https://llm.needl.tech/suites/1' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer <token>' \
--data-raw '{
    "name": "Edited Q & A Name",
    "required_inputs": ["d", "q"]
}'
```

</RequestExample>

<ResponseExample>

```json Response
{
  "id": 1,
  "org_id": "000000-0000-0000-0000000",
  "created_time": "00-00-00T00:00",
  "name": "Edited Q & A Name",
  "required_inputs": ["d", "q"]
}
```

</ResponseExample>
