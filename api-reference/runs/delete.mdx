---
title: "Delete a Run"
openapi: "DELETE /suites/{suite_id}/run/{run_id}"
description: "This endpoint deletes an existing suite. Deleting an existing suite deletes all associated test cases and runs."
---

### Body

<ParamField body="suite_id" type="number">
  The ID of the suite that should be deleted
</ParamField>

### Response

<ResponseField name="id" type="number">
  ID of the suite
</ResponseField>

<ResponseField name="org_id" type="string">
  Organization ID the suite belongs to
</ResponseField>

<ResponseField name="created_time" type="string">
  ISO formatted time the suite was created
</ResponseField>

<ResponseField name="name" type="Display name of the suite">
  ID of the suite
</ResponseField>

<ResponseField name="required_inputs" type="string[]">
  A list of the required inputs of the suite
</ResponseField>

<RequestExample>

```bash Example Request
curl --location --request DELETE 'https://llm.needl.tech/suites/1' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer <token>'
```

</RequestExample>

<ResponseExample>

```json Response
{
  "id": 1,
  "org_id": "000000-0000-0000-0000000",
  "created_time": "00-00-00T00:00",
  "name": "Q & A Search",
  "required_inputs": ["documents", "question"]
}
```

</ResponseExample>
